# Project Instructions

## Skills

Always apply the following skills when working on this project:

- **Frontend/UI work** (components, screens, pages, widgets): use the `frontend-design` skill and the `interface-design` skill. Prioritize distinctive, production-grade design — never generic AI aesthetics.
- **Error handling** (new features, APIs, async code): use the `error-handling-patterns` skill.
- **React/Vercel related code**: the `vercel-react-best-practices` skill applies.
